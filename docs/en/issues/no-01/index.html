Magazine Issue

No. 01

Before Choice

A first digital edition on taste forming in body and repetition before choice.

PublishedMarch 2026 FormatOnline edition / 4 scenes AccessOpen access

Editor's Note

The editor's note for this edition

The first issue treats taste as condition rather than result. It asks how tools, bodies, defaults, memory, and classification make choices possible.

The first issue of Habitus was built to read taste not as a list of likes but as bodily order shaped by repetition and experience.

01 / Opening Essay

The Body Is the Oldest Archive

Repeated experience leaves criteria for choice in the body.

The reason for choosing an object is often stored in the body long before language.

02 / Systems Note

Defaults Are Small Philosophy

It observes what a system does when the user does nothing.

Saying the user can change it is not enough. A good default considers the user's capacity to recover first.

03 / Tool File

Tools Remembered by the Hand

A good tool leaves hand position longer than new features.

Positions remembered by the hand are the slowest asset a brand builds.

Afterword / Editorial Desk / Closing

04 / Afterword

Repetition After Choice

A good choice makes repeatable life after purchase.

How something is kept longer forms the outline of taste more than what is chosen.
