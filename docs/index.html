About The Thing

The Thing은 서로 다른 분야를 하나의 편집 리듬으로 묶는 디지털 저널입니다.

예술의 이미지, 테크의 시스템, 디자인의 구조, 철학의 질문을 한 화면 안에서 연결합니다. 빠른 피드보다 분류의 밀도와 읽는 순서를 먼저 설계합니다.
