About the Magazine

Still Room은 서로 다른 분야를 하나의 편집 리듬으로 묶는 개인 웹매거진입니다.

예술의 이미지, 테크의 시스템, 디자인의 구조, 철학의 질문을 한 화면 안에서 연결합니다. 현재는 TypeScript 데이터로 글을 관리하고, 이후 CMS나 마크다운으로 확장할 수 있습니다.
