디자인 / 제품

표준이 되는 수납

무인양품의 수납용품은 장식보다 반복을 위해 설계된다. 반투명한 표면, 모듈의 치수, 이름을 앞세우지 않는 태도는 사용자가 물건을 고르는 방식보다 물건을 다시 넣는 방식을 바꾼다.

01 이름 없는 질서

수납의 좋은 디자인은 보관한 물건보다 다시 찾는 행동을 먼저 설계한다.

표준은 평범해 보이지만, 생활의 반복을 가장 오래 설계한다.

이름 없는 질서

무인양품의 수납용품과 수납 케이스 카테고리는 브랜드명이 강하게 앞서는 제품군이 아니다. 중요한 것은 표면이 얼마나 조용한지, 모듈이 얼마나 다시 쌓이는지, 방 안에서 얼마나 빨리 배경이 되는지다.

폴리프로필렌의 반투명함은 완전한 노출과 완전한 은폐 사이에 있다. 내용물은 흐리게 보이고, 사용자는 모든 것을 라벨링하지 않아도 대략의 위치를 기억한다.

수납이 생활을 바꾸는 방식은 극적이지 않다. 같은 높이의 박스가 쌓이고, 같은 손잡이가 반복되고, 비슷한 투명도가 방 안의 소음을 낮출 때 사용자는 물건보다 자신의 규칙을 먼저 보게 된다.

투명하지 않은 수납은 기억을 강요하고, 완전히 투명한 수납은 시각 소음을 만든다. MUJI의 반투명 표면은 그 사이에서 물건을 흐리게 남겨 두어 사용자가 위치를 짐작하되 방 전체가 목록처럼 보이지 않게 한다.

반복 가능한 치수

수납의 취향은 단품의 아름다움보다 여러 개가 놓였을 때 생기는 질서에서 드러난다. 같은 깊이와 폭이 반복될 때 방은 물건의 수보다 사용자의 규칙을 먼저 보여준다.

잘 만든 표준은 고르는 순간보다 되돌려 놓는 순간에 빛난다.

MUJI의 디자인은 선택의 흥분보다 유지의 안정감을 선택한다. 그래서 이 제품들은 새로 산 날보다 오랫동안 쓰이는 동안 더 강하게 존재감을 갖는다.

좋은 표준은 사용자가 매번 디자인을 의식하지 않아도 되도록 만든다. 반복되는 치수와 조용한 표면은 방의 결정을 줄이고, 줄어든 결정은 오래 쓰는 취향의 기반이 된다.

이 표준은 한 번의 정리보다 다음 정리의 가능성을 만든다. 사용자가 서랍을 열고 같은 모듈을 다시 추가할 수 있다는 감각은 제품을 소모품이 아니라 생활의 문법으로 만든다.
