No. 02

Rooms of Use

물건이 쓰이는 장소가 취향을 바꾸는 두 번째 디지털 에디션

발행2026년 4월 형식온라인 에디션 / 4개 장면 상태오픈 액세스

Editor's Note

이 호를 여는 편집 노트

두 번째 이슈는 매장, 집, 작업실, 여행 가방처럼 물건의 의미가 바뀌는 장소를 다룹니다. 같은 제품도 어디에 놓이는가에 따라 다른 취향이 됩니다.

물건의 의미는 스펙만으로 정해지지 않습니다. 손이 닿는 높이, 방의 조도, 이동하는 거리, 다시 넣는 위치가 취향을 만듭니다.

Contents

이 호의 장면

01 Cover Note 선반은 인터페이스다 좋은 선반은 물건을 보여주는 대신 다시 찾게 하는 순서를 만듭니다. 02 Object File 가방 안의 작은 방 무엇을 접고 무엇을 꺼내기 쉽게 둘지는 이동 중의 취향을 드러냅니다. 03 Spatial Essay 매장의 동선은 생활의 문법 어떤 순서로 걷게 하는지가 브랜드의 생활관을 드러냅니다. 04 Afterword 사용 이후의 취향 이번 호는 물건이 방과 이동 속에서 다시 의미를 얻는 장면으로 닫힙니다.

Space / Editorial Desk / Editorial

01 / Cover Note

선반은 인터페이스다

수납과 진열 사이에서 선반은 선택의 속도를 정합니다.

선반은 가구이면서 인터페이스입니다. 무엇이 앞에 오고 무엇이 뒤에 숨는지에 따라 생활의 우선순위가 달라집니다.

Object / Field notes / Travel

02 / Object File

가방 안의 작은 방

여행 가방은 이동을 위한 물건이면서 임시 방의 구조입니다.

가방은 공간의 축소판입니다. 내부 포켓, 어깨끈, 방수 소재는 이동의 불안을 줄이고 사용자가 임시로 머무는 방을 만듭니다.

Space / Route study / Store

03 / Spatial Essay

매장의 동선은 생활의 문법

매장은 제품을 파는 곳 이전에 생활을 분류하는 장치입니다.

분류와 동선은 보이지 않는 편집입니다. 매장 안에서 사용자는 브랜드가 정한 생활의 목차를 몸으로 읽습니다.

Afterword / Editorial Desk / Closing

04 / Afterword

사용 이후의 취향

취향은 구매 순간보다 사용 이후의 반복에서 더 잘 보입니다.

무엇을 좋아하는지는 구매보다 유지에서 드러납니다. 다시 놓고, 다시 충전하고, 다시 켜는 반복이 취향을 만듭니다.
