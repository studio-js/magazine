Magazine Issue

No. 01

Before Choice

취향이 선택 이전의 몸과 반복에서 만들어지는 첫 번째 디지털 에디션

발행2026년 3월 형식온라인 에디션 / 4개 장면 상태오픈 액세스

Editor's Note

이 호를 여는 편집 노트

첫 번째 이슈는 취향을 결과가 아니라 조건으로 봅니다. 도구, 몸, 기본값, 기억, 분류가 어떤 선택을 가능하게 만드는지 묻습니다.

Habitus의 첫 이슈는 취향을 좋아함의 목록이 아니라 반복과 경험이 만든 몸의 질서로 읽기 위해 설계되었습니다.

Contents

이 호의 장면

01 Opening Essay 몸은 가장 오래된 아카이브 취향은 설명보다 먼저 자세와 속도에 남습니다. 02 Systems Note 기본값은 작은 철학 기본값은 선택 이전의 선택입니다. 03 Tool File 손이 기억하는 도구 기능보다 조작의 안정감이 취향을 만듭니다. 04 Afterword 선택 이후의 반복 취향은 한 번의 선택보다 선택 이후의 관리에서 더 오래 남습니다.

Essay / Habitus / Editorial

01 / Opening Essay

몸은 가장 오래된 아카이브

반복된 경험은 몸에 선택의 기준을 남깁니다.

어떤 물건을 고르는 이유는 말보다 오래전 몸에 저장되어 있을 때가 많습니다.

Systems / Editorial Desk / Interface

02 / Systems Note

기본값은 작은 철학

아무것도 하지 않았을 때 시스템이 무엇을 하는지 봅니다.

사용자가 바꿀 수 있다는 말은 충분하지 않습니다. 좋은 기본값은 사용자의 회복 가능성을 먼저 생각합니다.

Tools / Field notes / Studio

03 / Tool File

손이 기억하는 도구

좋은 도구는 새 기능보다 손의 위치를 오래 남깁니다.

손이 기억하는 위치는 브랜드가 가장 느리게 쌓는 자산입니다.

Afterword / Editorial Desk / Closing

04 / Afterword

선택 이후의 반복

좋은 선택은 구매가 끝난 뒤 반복 가능한 생활을 만듭니다.

무엇을 고르는가보다 그것을 어떻게 오래 쓰는지가 취향의 윤곽을 만듭니다.
